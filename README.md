# Three.js 案例集合

这是一个 Three.js 案例集合项目，展示各种 Three.js 的使用示例和效果。

## 项目结构

- `demos/` - 存放所有案例，每个案例有自己独立的文件夹
- `index.html` - 项目首页，展示所有案例的列表
- `main.js` - 首页的主要脚本文件
- `style.css` - 首页的样式文件

## 如何使用

1. 安装依赖：
   ```
   npm install
   ```

2. 启动开发服务器：
   ```
   npm run dev
   ```

3. 构建项目：
   ```
   npm run build
   ```

## 添加新案例

1. 在 `demos` 目录下创建一个新的文件夹，命名为你的案例名称
2. 在新文件夹中创建必要的文件（如 `index.html`, `main.js` 等）
3. 在 `demos/index.js` 中添加你的案例信息
